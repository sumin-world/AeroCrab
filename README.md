## ⚡ 빌드 & 실행 예시

### 컴파일

```bash
gcc -O2 -Wall -std=c11 -o main boj/1000_A+B.c
```

### 실행

```bash
./main < input.txt
```

### 디버그 모드

```bash
gcc -g -Wall -std=c11 -o debug boj/1000_A+B.c
gdb ./debug
```
